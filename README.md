# Install
```
npm install nulll
```

# Usage
```
const nulll = require("nulll");

console.log(nulll); // print "null"
```
Nulll helps you to reach your target LOC. Comes with only 5 unreviewed dependencies and a tinny footprint of less than 400MB.
